Pipeline¶

class darts.dataprocessing.pipeline.Pipeline(transformers: Sequence[darts.dataprocessing.transformers.base_data_transformer.BaseDataTransformer[darts.timeseries.TimeSeries]], copy: bool = False)[source]¶

Bases: object

Pipeline combines multiple data transformers chaining them together.

Parameters

  • transformers – Sequence of data transformers.

  • copy – If set makes a (deep) copy of each data transformer before adding them to the pipeline

fit(data: darts.timeseries.TimeSeries)[source]¶

Fit all fittable transformers in pipeline.

Parameters

data – TimeSeries to fit on.

fit_transform(data: darts.timeseries.TimeSeries) → darts.timeseries.TimeSeries[source]¶

For each data transformer in pipeline first fit the data if transformer is fittable then transform data using fitted transformer. The transformed data is then passed to next transformer.

Parameters

data – TimeSeries to fit and transform on.

Returns

Transformed data.

Return type

TimeSeries

inverse_transform(data: darts.timeseries.TimeSeries) → darts.timeseries.TimeSeries[source]¶

For each data transformer in pipeline inverse_transform data. Then inverse transformed data is passed to next transformer. Transformers are traversed in reverse order. Raises value error if not all of the transformers are invertible.

Parameters

data – TimeSeries to be inverse transformed.

Returns

Inverse transformed data.

Return type

TimeSeries

invertible() → bool[source]¶

Returns whether the pipeline is invertible or not. A pipeline is invertible if all transformers in the pipeline are themselves invertible.

Returns

True if the pipeline is invertible, False otherwise

Return type

bool
