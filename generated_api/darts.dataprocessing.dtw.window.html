class darts.dataprocessing.dtw.window.CRWindow(n: int, m: int, ranges: Optional[numpy.ndarray] = None)[source]¶

Bases: object

Compressed row representation window. Stores the range of active grid cells in each column. Any window with contiguous columns can be expressed as an CRWindow. Supports efficient iterative construction and updates.

Parameters

  • n – The width of the window, must be equal to the length of series1

  • m – The height of the window, must be equal to the length of series2

  • ranges – Ranges of active cells within a column [[start_column0, end_column0], …] with shape (n, 2) and where start >= 0 and end <= m.

add_range(column: int, start: int, end: int)[source]¶

Extends the active cells in the column by the range (start,end). Ranges smaller than the current one are ignored. Note (1, m+1), not (0,m) corresponds to an entire column.

  • end – Row element int index where end >= 1 and end <= m+1
